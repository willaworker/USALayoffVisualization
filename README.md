# **项目简介**

#### 项目介绍：

​	BIT可视化课程项目。近些年一直有关于科技公司裁员的报道，故寻找相关数据集进行可视化，更加形象直观的展现美国公司裁员的数量、比例与公司、地区、行业、规模和融资阶段的关系，使用vue.js和echarts进行页面制作。
